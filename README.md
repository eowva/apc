# apc
This is the repository for a mmWave-scanner based Automated Passenger Counter (APC) device. Built as a final project for the College Park Scholars Data Justice program.

Version 1 (Last updated 2025-05-01) 

# Components
 - LD-2450 mmWave sensor
 - ESP32 Wi-Fi board
 - 18650 Power Source
 - 3-D Printed housing

# General Structure
The device is capable of registering boards and deboards based on the direction of a subject's travel, by connected the LD-2450 sensor to the ESP board. This data is saved to a CSV file that is accessible through a locally hosted webpage.
